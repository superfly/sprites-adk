# ADK docs catalog submission

Staging for the [google/adk-docs](https://github.com/google/adk-docs/blob/main/CONTRIBUTING.md#integrations) integrations catalog. To submit, copy into a fork of `google/adk-docs`:

- `sprites.md` → `docs/integrations/sprites.md`
- `assets/sprites.png` → `docs/integrations/assets/sprites.png`
